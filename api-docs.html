介绍

OpenClaw API 基于 Flask 构建，提供 RESTful 接口。所有响应均为 JSON 格式。

Base URL
https://aiintel.top/api

所有 API 端点都返回统一的响应格式：

响应格式
{
  "success": true,
  "timestamp": "2026-04-15T12:00:00",
  "data": { ... }
}

认证

大多数 API 端点无需认证。受保护的端点需要在请求头中添加 API 密钥。

请求头
X-API-Key: your_api_key_here
Content-Type: application/json

错误码

HTTP 状态码说明处理建议
200成功正常处理响应
400请求参数错误检查请求体和参数
401未授权检查 API 密钥
403禁止访问检查权限设置
404资源不存在检查 URL 路径
429请求过于频繁降低请求频率
500服务器内部错误稍后重试
POST /api/chat

发送聊天消息到 OpenClaw，获取 AI 回复。使用共享会话记忆。

请求参数

参数类型必填说明
messagestring必填用户消息内容
user_namestring可选用户名（用于会话区分）
modestring可选模式: normal / agent (默认: normal)
请求示例
// curl 示例
curl -X POST https://aiintel.top/api/chat \
  -H "Content-Type: application/json" \
  -d '{
    "message": "你好，OpenClaw！",
    "user_name": "king",
    "mode": "normal"
  }'

响应

200 OK
{
  "success": true,
  "response": "你好！我是 OpenClaw 🦞",
  "session_id": "session_shared_normal",
  "duration_ms": 1234,
  "model": "qwen3-max-2026-01-23",
  "timestamp": "2026-04-15T12:00:00",
  "mode": "normal"
}
200 OK 400 Bad Request 500 Error
POST /api/chat/stream

流式聊天接口，使用 Server-Sent Events (SSE) 返回逐字输出。

SSE 数据格式
// 文本片段
data: {"delta": "你好", "mode": "normal"}

// 完成信号
data: {"done": true, "duration_ms": 2345}

// 心跳包
data: {"heartbeat": true}

GET /api/status

获取系统运行状态，包括 CPU、内存、运行时间等信息。

200 OK 响应
{
  "success": true,
  "core_status": "running",
  "status": "running",
  "uptime": 86400,
  "dream_phase": "idle",
  "system": {
    "cpu_percent": 23.5,
    "memory_used": 4096,
    "memory_total": 8192,
    "memory_percent": 50.0
  },
  "timestamp": "2026-04-15T12:00:00"
}
GET /api/stats

获取项目统计信息：代码行数、技能数、工具数、画作数量等。

响应示例
{
  "success": true,
  "total_lines": 125000,
  "tools_count": 26,
  "skill_count": 200,
  "memory_files": 15,
  "file_count": 450,
  "painting_count": 123,
  "timestamp": "2026-04-15T12:00:00"
}
GET /api/health

健康检查端点，用于监控和负载均衡。

响应示例
{
  "success": true,
  "status": "healthy",
  "version": "3.1.0",
  "timestamp": "2026-04-15T12:00:00"
}
GET /api/components

获取 3.1 核心组件的运行状态。

GET /api/modules

获取核心模块列表和状态。

POST /api/gallery/update

触发画廊更新，检测新画作并自动更新网站。

POST /api/gallery/daily

触发每日创作，生成 10 幅新画并更新网站。

POST /api/desktop

AI 桌面控制：截图、鼠标、键盘、窗口管理。

GET /api/vnc/status

检查虚拟桌面 (VNC) 运行状态。
